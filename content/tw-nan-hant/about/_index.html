鳴謝

本站由香港語言學學會 (LSHK) 支持建成。

感謝 LSHK 粵拼小組佮粵典對本站的支持。特別感謝 Chaak-ming Lau 博士，Kwan-hin Cheung 博士，Andy Chin 博士，Raymond Tze 先生，Muhan Zhong 先生佮 Zungman Ceoi 先生為著建設本站所提供的幫助。

感謝具區句魚先生提供本站吳語版本。

開發佮貢獻

你若想欲支持粵拼抑是參與阮的事務，你會當：

  • 寄批予阮知影你的想法抑是意見。
  • 佇 GitHub 頂面開一个新的 issue 討論抑是代碼 pull request。
你若會曉粵語佮其他語言，而且願意共這个網站翻譯做其他語言，請你著寄批聯絡阮，阮真歡迎你的幫助佮貢獻。
