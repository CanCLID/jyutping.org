鳴謝

本站由粵語計算語言學基礎建設組建成，並得到香港語言學學會 (LSHK) 嘅協助。網站原始碼全部開源喺GitHub，歡迎外界貢獻。

感謝 LSHK 粵拼小組同粵典對本站嘅支持。特別鳴謝 Chaak-ming Lau 博士、Andy Chin 博士、Kwan-hin Cheung 博士、Raymond Tze 先生同 Zungman Ceoi 先生爲建設本站提供嘅幫助。

感謝 Lim Hian-tong 先生提供本站臺灣閩南語版本。

感謝具區句魚先生提供本站吳語版本。

感謝以成提供本站越南語版本。

開發及貢獻

若果閣下想支持粵拼或者參與我哋嘅事務，你可以：

  • 傳送電郵至 support@jyutping.org 話畀我哋知你嘅諗法或者回饋意見。
  • 喺 GitHub 上面新開一個 issue 討論或者代碼 pull request。
  • 安裝同使用粵拼輸入法嚟打字，以及同人分享本網站。
若果閣下識得粵語，並且願意將本網站翻譯成其他語言，請務必以電郵聯絡我哋，我哋非常歡迎你嘅幫助同貢獻。
