鳴謝

本站由香港語言學學會 (LSHK) 支持建成。

感謝 LSHK 粵拼小組同粵典對本站嘅支持。特別鳴謝 Chaak-ming Lau 博士，Andy Chin 博士，Raymond Tze 先生，Muhan Zhong 先生同 Zungman Ceoi 先生爲建設本站提供嘅幫助。

感謝 Lim Hiantong 先生提供本站臺灣閩南語版本。

開發及貢獻

如果你想支持粵拼或參與到我哋嘅事務中，你可以：

  • 發郵件話畀我哋知你嘅諗法或者反饋意見。
  • 喺 GitHub 上面新開一個 issue 討論或者代碼 pull request。
如果你識得粵語同其他語言，并願意將本網站翻譯成其他語言，請務必郵件聯繫我哋，我哋非常歡迎你嘅幫助同貢獻。
