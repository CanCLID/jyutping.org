鳴謝

本站由香港語言學學會 (LSHK) 支持建成。

感謝 LSHK 粵拼小組對本站的支持。特別鳴謝 Chaak-ming Lau 博士，Andy Chin 教授，Muhan Zhong 先生和 Zungman Ceoi 先生爲建設本站提供的幫助。

開發及貢獻

如果你想支持粵拼或參與到我們的事務中，你可以：

  • 發郵件告訴我們你的想法或者反饋意見。
  • 喺 GitHub 上面新開一個 issue 討論或者代碼 pull request。
如果你熟習粵語和其他語言，并願意把本網站翻譯成其他語言，請務必郵件聯繫我們，我們非常歡迎你的幫助和貢獻。
