{{< alert title="方案選字原則">}}

本用字方案對於句末語氣詞從分弗從合，即儘量使用**一隻漢字對應一隻語素**。對於音節相同个字，一般將第 1、3、4 聲定爲獨立个語素，對於第 2、5、6 聲視爲變調，個咾**推薦使用標點符號**來表示，弗再另外指定漢字。譬方我伲推薦用「㗎？」來代表 **gaa2**，但弗用「𠿪」。

對於大部分 **aa** 開音節語氣詞，本方案推薦弗區分 **aa** 同 **a**，譬方 **laa3** 搭 **la3** 看成功一隻語素。
{{< /alert >}}

## 句末語氣詞推薦用字表

{{< alert title="說明">}}

- 「了」字僅僅讀作 **liu5**，弗推薦作爲句末語氣詞使用。
- 粵語裏向無麼「吧」迭隻語氣詞，「吧」只有官話書面語搭普通話/國語裏向有。
- 「嗱」**naa4** 作爲句首感嘆詞，也加到仔表裏向，以表示搭句末語氣詞「喇」**laa4** 區別。

## 組合語氣詞示例

{{< alert title="說明">}}

下表是粵語中部分組合語氣詞示例。注意組合語氣詞當中个非句末字發音長度作興比一般性情況短着眼，譬如「吖嘛」中个「吖」好讀 **aa1** 或 **a1**。
{{< /alert >}}

| 漢字        | 拼音                   | 例句                                                    | 備註                             |
| ----------- | ---------------------- | ------------------------------------------------------- | -------------------------------- |
| 吖嘛        | aa1 maa3 / a1 maa3     | 聽日係星期日～。                                        |                                  |
| 啊嘛        | aa6 maa5 / a6 maa5     | 唔係噉～？                                              |                                  |
| 哎吖吖      | ai1 aa1 aa1            | ～，做乜搞成噉啊？                                      | 「哎」字好讀 ai1 或 aai1，下同。 |
| 哎吔/哎吔吔 | ai1 jaa1/ai1 jaa1 jaa1 | ～！豈有此理！                                          |                                  |
| 哎呀        | ai1 aa4/ai1 jaa5       | ～，你唔係噉啊嘛？                                      |                                  |
| 𠺢嘛        | gaa1 maa3/ga1 maa3     | 噉我唔知～！                                            | 「𠺢」次選「咖」                 |
| 㗎咩        | gaa3 me3/ ga3 me3      | 今日唔係放假～？                                        |                                  |
| 㗎嘛        | gaa6 maa5 / ga6 maa5   | 唔係噉～？                                              |                                  |
| 嘅啫        | ge3 ze1                | 無非就係佢一個人～。                                    |                                  |
| 嘞噃        | la3 bo3 / lak3 bo3     | 你好～，唔好再得寸進尺～！                              |                                  |
| 吓話        | haa6 waa5 / ha6 waa5   | 1. 係得咁多～？<br><br> 2. 「呢隻股票實升㗎！」「～？」 |                                  |
| 喇啩        | laa3 gwaa3 / la3 gwaa3 | 噉應該搞掂～？                                          |                                  |
| 㗎喇噃      | ga3 la3 bo3            | 你再唔嚟就食晒～。                                      |                                  |
