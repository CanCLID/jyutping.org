{{< alert title="寫法區別 & 音系區別">}}
粵拼脫吳拼個區別分兩類：寫法區別脫仔音系區別。

**寫法區別**
就是搿個發音辣吳語脫粵語裏向儕有個，只不过辣兩種拼音裏向用弗同箇字母來代表，搿一類區別一看就懂。

**音系區別**
就是吳話裏向嘸沒搿個發音，僅僅是粵語特有個，搿一類往往也是吳話母語者學粵語個重點難點。
{{< /alert >}}

> 本文辣講解發音個時候會得用到國際音標（IPA），如果讀者弗熟悉國際音標也嘸沒關係，直接點擊圖標 {{< audio >}} 聽例字個錄音就可以感受到具體個發音唻。
>
> 本文中所有粵拼儕採用粗體顯示。每個例字拼音後面個數字 1 - 6 是聲調，讀者可以暫時先忽略，或者直接跳到 [2.7 聲調](#2-7-聲調) 一節瞭解好聲調之後再開始閱讀本文。

## 1. 寫法區別

### 1.1 吳拼 i/y → 粵拼 j

{{< alert color="success">}}
吳拼個 i/y 辣粵拼裏向寫成功 **j**。粵拼寫成功 **j** 是出於兩個考慮：第一是搿個發音辣國際音標裏向就是 [j]（吳拼個 y 是i個濁音）；第二是如果寫成功 y 會得脫韻母 **yu** 混淆。
{{< /alert >}}

| 例字                       | ✔️ 正確寫法 | ❌ 錯誤寫法 |
| -------------------------- | ----------- | ----------- |
| 一 {{< audio id="jat1" >}} | jat1        | iat1        |
| 以 {{< audio id="ji5" >}}  | ji5         | i5         |
| 夜 {{< audio id="je6" >}}  | je6         | ye6         |

### 1.2 吳拼 gu-、khu、ku- → 粵拼 gw-、kw-

{{< alert color="success">}}
吳拼個 gu-、khu-,ku-，辣粵拼裏向用 **gw-**、**kw-** 來表示。
{{< /alert >}}

| 例字                         | ✔️ 正確寫法 | ❌ 錯誤寫法 |
| ---------------------------- | ----------- | ----------- |
| 廣 {{< audio id="gwong2" >}} | gwong2      | guong2      |
| 狂 {{< audio id="kwong4" >}} | kwong4      | khuong4      |
| 昆 {{< audio id="kwan1" >}}  | kwan1       | khuan1       |
| 古 {{< audio id="gu2" >}}    | gu2         | gwu2 / gw2  |

### 1.3 吳拼 iu → 粵拼 yu

{{< alert color="success">}}
搿個音國際音標是 [y]。搿個韻母雖然是單元音，但是辣粵拼裏向用了 **yu** 兩個字母來表示（吳拼是iu）。可以避免脫前頭講到個聲母 **j** 混淆。
{{< /alert >}}

| 例字                        | ✔️ 正確寫法 | ❌ 錯誤寫法 |
| --------------------------- | ----------- | ----------- |
| 豬 {{< audio id="zyu1" >}}  | zyu1        | ziu1         |
| 村 {{< audio id="cyun1" >}} | cyun1       | ciun1        |
| 雨{{< audio id="jyu5" >}}   | jyu5        | jiu5 / yu5   |

### 1.4 吳拼 on → 粵拼 ung

{{< alert color="success">}}
吳拼裏向個 on 脫仔粵拼個 **ung** 發音是一樣個，國際音標儕是 [ʊŋ]，粵拼裏向個 **ong** 脱湖州言話裏向“當”字個韻母是一樣個，國際音標爲 [ɔŋ]。
{{< /alert >}}

| ung                               | ong                               |
| --------------------------------- | --------------------------------- |
| 鍾 zung1 {{< audio id="zung1" >}} | 莊 zong1 {{< audio id="zong1" >}} |
| 公 gung1 {{< audio id="gung1" >}} | 江 gong1 {{< audio id="gong1" >}} |

### 1.5 吳拼 au → 粵拼 aau

{{< alert color="success">}}
粵拼個 aau 脫吳語個 au 是一樣個，但是粵拼個 au 脫吳拼個 au 弗一樣
{{< /alert >}}

| 例字                        | ✔️ 正確寫法 | ❌ 錯誤寫法  |
| --------------------------- | ----------- | ------------ |
| 考 {{< audio id="haau2" >}} | haau2       | hau2 |
| 包 {{< audio id="baau1" >}} | baau1       | bau1 |

> 關於粵拼中 **a** 和 **aa** 的區別下面會講解。

## 2. 音系區別

### 2.1 韻腹 aa 和 a

{{< alert color="success">}}
搿是吳語話母語者學習粵語個最大難點之一。吳拼個 a 等同於粵拼的 **aa**，國際音標是 [a]。粵拼裏向個 **a** ，伊個國際音標是 [ɐ]。（新派上海言話“達”字個韻母是[ɐʔ]）。 **aa** 脫 **a** 是兩個弗同個韻腹，弗可以混淆。
{{< /alert >}}

| 例字                              |                                   |                                   |
| --------------------------------- | --------------------------------- | --------------------------------- |
| 三 saam1 {{< audio id="saam1" >}} | 山 saan1 {{< audio id="saan1" >}} | 稍 saau2 {{< audio id="saau2" >}} |
| 心 sam1 {{< audio id="sam1" >}}   | 身 san1 {{< audio id="san1" >}}   | 手 sau2 {{< audio id="sau2" >}}   |

### 2.2 韻腹 eo、oe

{{< alert color="success">}}
搿兩個韻母個國際音標分別是 [ɵ]、[œ]，類似上海言話裏向“安”個發音。
{{< /alert >}}

| 例字                              |                                     |
| --------------------------------- | ----------------------------------- |
| 率 seot1 {{< audio id="seot1" >}} | 潤 jeon6 {{< audio id="jeon6" >}}   |
| 削 soek3 {{< audio id="soek3" >}} | 讓 joeng6 {{< audio id="joeng6" >}} |

### 2.3 韻母 e

{{< alert color="success">}}
吳拼個 e 的國際音標是 [e]，脫粵拼裏向個 **e** （國際音標是 [ɛ]）比較接近，類似英語單詞 「yes」 個元音。
{{< /alert >}}

- 寫 se2 {{< audio id="se2" >}}
- 借 ze3 {{< audio id="ze3" >}}
- 鄭 zeng6 {{< audio id="zeng6" >}}
- 石 sek6 {{< audio id="sek6" >}}

### 2.4 韻母 i

{{< alert color="success">}}
吳拼裏向個 i 代表的音是 [i]。粵拼個 **i** ，辣大多數辰光代表發音 [i]，辣海拼寫成功 **ik** 或者 **ing** 個辰光，發音是 [ɪ]。留意對比下面例字個發音。
{{< /alert >}}

| [i]                             | [ɪ]                               |
| ------------------------------- | --------------------------------- |
| 詩 si1 {{< audio id="si1" >}}   | 識 sik1 {{< audio id="sik1" >}}   |
| 仙 sin1 {{< audio id="sin1" >}} | 星 sing1 {{< audio id="sing1" >}} |
| 知 zi1 {{< audio id="zi1" >}}   | 即 zik1 {{< audio id="zik1" >}}   |

### 2.5 聲母 ng-

{{< alert color="success">}}
個隻音個國際音標是 [ŋ]。粵拼是ng，脫吳拼一樣。ng聲母辣吳語當中是非常常見個，譬如“硬” “牙” 等字。不過吳語裏向声母是ng個字脱粵語裏向個弗是完全一样個。
{{< /alert >}}

| 例字                                |                                 |
| ----------------------------------- | ------------------------------- |
| 岸 ngon6 {{< audio id="ngon6" >}}   | 按 on3 {{< audio id="on3" >}}   |
| 雁 ngaan6 {{< audio id="ngaan6" >}} | 晏 aan3 {{< audio id="aan3" >}} |

### 2.6 韻尾 -m

{{< alert color="success">}}
吳語裏向個鼻音韻尾只有 -n 和 -ng，嘸沒 -m。但是粵語有得 **-m**、**-n**、**-ng** 三種鼻音韻尾。
{{< /alert >}}

| 例字                                |                                  |                                   |
| ----------------------------------- | -------------------------------- | --------------------------------- |
| 三 saam1 {{< audio id="saam1" >}}   | 含 ham4 {{< audio id="ham4" >}}  | 尖 zim1 {{< audio id="zim1" >}}   |
| 山 saan1 {{< audio id="saan1" >}}   | 痕 han4 {{< audio id="han4" >}}  | 煎 zin1 {{< audio id="zin1" >}}   |
| 生 saang1 {{< audio id="saang1" >}} | 行 hang4{{< audio id="hang4" >}} | 精 zing1 {{< audio id="zing1" >}} |

### 2.7 韻尾 -p、-t、-k

{{< alert color="success">}}
搿三隻韻尾是粵語裏向個「入聲」。吳語裏向個入聲只有一種[ʔ]。粵語裏向個入聲字辣辣吳語裏向基本浪也是入聲字。

粵語裏的入聲韻尾 **-p**、**-t**、**-k** 特別像英語單詞 cap、put、book 末尾個輔音，但有個最大個區別，粵語個入聲韻尾儕是弗除阻個（國際音標分別是 [p̚]、[t̚]、[k̚]），也就是辣音節最後弗會得像英語一樣呼出空氣。

{{< /alert >}}

| -p                                | -t                              | -k                              |
| --------------------------------- | ------------------------------- | ------------------------------- |
| 湿 sap1 {{< audio id="sap1" >}}   | 失 sat1 {{< audio id="sat1" >}} | 塞 sak1 {{< audio id="sak1" >}} |
| 接 zip3 {{< audio id="zip3" >}}   | 节 zit3 {{< audio id="zit3" >}} | 脊 zek3 {{< audio id="zek3" >}} |
| 集 zaap6 {{< audio id="zaap6" >}} | 活 wut6 {{< audio id="wut6" >}} | 六 luk6 {{< audio id="luk6" >}} |

### 2.8 聲調

{{< alert color="dark">}}
如果講儂只不過想學會用粵拼打字，可以**直接跳過本節**。粵語拼音輸入法是弗需要打聲調個，但是如果加上聲調個言話能夠提高輸入速度。
{{< /alert >}}

粵語有得六個聲調：

粵語個 6 個聲調如下圖：

![image](/img/yue.png)

6 個調值如下表：

| 第 1 聲 55（高平調）          | 第 2 聲 35（高升調）          | 第 3 聲 33（中平調）          |
| ----------------------------- | ----------------------------- | ----------------------------- |
| 詩 si1 {{< audio id="si1" >}} | 史 si2 {{< audio id="si2" >}} | 試 si3 {{< audio id="si3" >}} |

| 第 4 聲 21（低降調）          | 第 5 聲 13（低升調）          | 第 6 聲 22（中低平調）        |
| ----------------------------- | ----------------------------- | ----------------------------- |
| 時 si4 {{< audio id="si4" >}} | 市 si5 {{< audio id="si5" >}} | 事 si6 {{< audio id="si6" >}} |

粵語個第一聲脫常州言話個陰平調調值是一樣個，第二聲脫無錫言話個陰去調一樣個。第三聲脫崇明言話個陰去調一樣個。第五聲脫臨平言話個陽平調一樣，第六聲脫桐鄉個陽平調一樣。（辣吳語區裏向嘸沒尋到脱粤语第四聲調值一樣個例子）初學者需要較多個聽力訓練來增強辨析。阿拉提供了一個練習聲調個練習頁，來撥讀者反覆鍛鍊辨析粵語聲調：[聲調辨別練習](/zh-cmn-hant/docs/tone)。

練習粵語聲調還有一個好孛相又有得效果個方法，就是利用粵語歌，詳見本站[用粵語歌練習聲調](/zh-cmn-hant/blog/song/)。
