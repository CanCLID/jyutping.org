## 1 粤拼とは？

粤拼（ユッピン）は、広東語の単語をアルファベット（a-z）と数字（1-6）だけで表記する方法です。日本語話者であれば、広東語の発音をマスターするために覚える必要がある記号はほんの一握りです。

英語では、同じ文字が複数の方法で発音されることがあります。最も極端な例を見てみましょう：-ough。この4文字は cough、though、through、thought、borough でまったく異なる音になります。**英語のスペルは非常に不規則で一貫性がありません**。

粤拼は英語とは異なり、規則的になるよう設計されたシステムです。各音（専門用語では音素）を表記する方法は一つだけであり、各文字に対応する音も一つだけです。言い換えれば、粤拼は広東語の音からラテン文字への**一対一の対応規則**を提供します。

これは学習者が言語をマスターするのを助けるシステムです。広東語と日本語は非常に異なる言語であることを念頭に置いてください。広東語は発音の面ではそれほど難しくありませんが（他で聞いたことがあるかもしれませんが）、取り組む必要があることがいくつかあります。粤拼はそれほど難しくありませんが、以下の3つのことに取り組む必要があります：

1. **見慣れない綴り**：一部の文字や綴りの規則が馴染みのないものに見えるかもしれません。
2. **微妙な違い**：発音は分かりやすそうに見えても、よりネイティブに近い音にするために少し調整が必要な場合があります。
3. **新しい音**：すべてが簡単とは言いません。まったく新しい音があります：今まで聞いたことのない奇妙な音、耳にとってまったく新しい区別があります。

心配しないでください。このガイドがあります。

従来の教科書とは異なり、長い音の対応表は提供しません（必要であれば[こちら](/jp/jyutping)で見つけることができます）。それは逆効果です。学ぶ必要があることを上記の3つの異なるタイプに分解していきます。

> 音を説明する際、IPA（国際音声記号）を提供します。IPAが何か分からない場合は、スキップしてください。すべての粤拼の綴りは太字で表示されます。ボタンをクリックしてサンプル音を聞き、音を真似してみてください。

## 2 見慣れない綴り

まず母音を見てみましょう。広東語の母音の綴りはかなり規則的なので、一つずつ見ていきましょう。日本語のローマ字とは対応しないため、見慣れない綴りのカテゴリに入ります。

### 2.1 母音 **aa** と **a**

<Alert color="success">
  **aa** は長い「あ」の音です。感嘆詞「あああああ！」は広東語で「aa」になります。
</Alert>

**a** は短い「あ」の音で、日本語の「あ」より少し狭い音です。以下の例を聞いてください。

| aa                                        | a                                       |
| ----------------------------------------- | --------------------------------------- |
| 三 saam1 <Audio client:load id="saam1" /> | 心 sam1 <Audio client:load id="sam1" /> |
| 山 saan1 <Audio client:load id="saan1" /> | 身 san1 <Audio client:load id="san1" /> |
| 稍 saau2 <Audio client:load id="saau2" /> | 手 sau2 <Audio client:load id="sau2" /> |

### 2.2 母音 **i**

<Alert color="success">
  日本語の「い」に似ていますが、IPAは[i]です。ただし、**i** の後に **-k** または **-ng** が続く場合は例外があります：**-ik** は英語の -ick のように、**-ing** は英語の -ing のように聞こえます。IPAは[ɪ]になります。以下の例を聞いてください。
</Alert>

| [i]                                     | [ɪ]                                       |
| --------------------------------------- | ----------------------------------------- |
| 詩 si1 <Audio client:load id="si1" />   | 識 sik1 <Audio client:load id="sik1" />   |
| 仙 sin1 <Audio client:load id="sin1" /> | 星 sing1 <Audio client:load id="sing1" /> |
| 知 zi1 <Audio client:load id="zi1" />   | 即 zik1 <Audio client:load id="zik1" />   |

### 2.3 母音 **e**

<Alert color="success">
  日本語の「え」に近い音ですが、少し広い口で発音します。IPAは[ɛ]です。
</Alert>

- 寫 se2 <Audio client:load id="se2" />
- 借 ze3 <Audio client:load id="ze3" />
- 鄭 zeng6 <Audio client:load id="zeng6" />
- 石 sek6 <Audio client:load id="sek6" />

### 2.4 母音 **o**

<Alert color="success">
  日本語の「お」に近い音ですが、口をもっと丸くして発音します。IPAは[ɔ]です。以下の例を聞いてください：
</Alert>

- 歌 go1 <Audio client:load id="go1" />
- 可 ho2 <Audio client:load id="ho2" />
- 過 gwo3 <Audio client:load id="gwo3" />
- 博 bok3 <Audio client:load id="bok3" />
- 裝 zong1 <Audio client:load id="zong1" />

### 2.5 母音 **u**

<Alert color="success">
  日本語の「う」に似ていますが、唇をもっと丸くして発音します。IPAは[u]です。ただし、**-uk** または **-ung** では少し短くなります。
</Alert>

| [u]                                   | [ʊ]                                       |
| ------------------------------------- | ----------------------------------------- |
| 夫 fu1 <Audio client:load id="fu1" /> | 覆 fuk1 <Audio client:load id="fuk1" />   |
| 姑 gu1 <Audio client:load id="gu1" /> | 工 gung1 <Audio client:load id="gung1" /> |

### 2.6 子音 **j**

<Alert color="success">
  これは日本語の「や」行の子音に相当します。IPAは[j]です。例を見てください：
</Alert>

- 一 jat1 <Audio client:load id="jat1" />
- 以 ji5 <Audio client:load id="ji5" />
- 夜 je6 <Audio client:load id="je6" />

> なぜ粤拼は **y** ではなく **j** を使ってこの音 [j] を表すのでしょうか？それは、y を [j] に使うと、音 [y] を表す文字がなくなってしまうからです。この音については[母音 **yu**](#41-母音-yu)を参照してください。

## 3 微妙な違い

### 3.1 有声音と無気音・有気音

<Alert color="success">
  広東語には「有声破裂音」がないため、これらの音は日本語や英語の対応する音より少し柔らかくなります：b、d、g。技術的には、粤拼の **b-**、**d-**、**g-** は無気音（IPAは[p]、[t]、[k]）で、英語の spy、stay、sky の p、t、k と同じです。
</Alert>

一方、広東語には有気破裂音があります：**p-**、**t-**、**k-**。IPAは[pʰ]、[tʰ]、[kʰ]です。以下の例を聞いてください：

| 無気音                                  | 有気音                                  |
| --------------------------------------- | --------------------------------------- |
| 巴 baa1 <Audio client:load id="baa1" /> | 趴 paa1 <Audio client:load id="paa1" /> |
| 打 daa1 <Audio client:load id="daa1" /> | 他 taa1 <Audio client:load id="taa1" /> |
| 家 gaa1 <Audio client:load id="gaa1" /> | 卡 kaa1 <Audio client:load id="kaa1" /> |

### 3.2 内破音の語末破裂音

<Alert color="success">
  語末の **-p**、**-t**、**-k** は内破音で、IPAは[p̚]、[t̚]、[k̚]です。ネイティブスピーカーは破裂を「飲み込んで」そこで止めます。以下の例を聞いてください。「内破音」バージョンを真似できなくても大丈夫です。完全に理解されます。
</Alert>

| -p                                        | -t                                      | -k                                      |
| ----------------------------------------- | --------------------------------------- | --------------------------------------- |
| 濕 sap1 <Audio client:load id="sap1" />   | 失 sat1 <Audio client:load id="sat1" /> | 塞 sak1 <Audio client:load id="sak1" /> |
| 接 zip3 <Audio client:load id="zip3" />   | 節 zit3 <Audio client:load id="zit3" /> | 脊 zek3 <Audio client:load id="zek3" /> |
| 集 zaap6 <Audio client:load id="zaap6" /> | 活 wut6 <Audio client:load id="wut6" /> | 六 luk6 <Audio client:load id="luk6" /> |

## 4 新しい音

### 4.1 母音 **yu**

<Alert color="success">
  「い」を発音しながら同時に唇を丸めてみてください。これはドイツ語の「ü」と同じ音で、IPAは[y]です。以下の例を聞いてください：
</Alert>

- 豬 zyu1 <Audio client:load id="zyu1" />
- 村 cyun1 <Audio client:load id="cyun1" />
- 雨 jyu5 <Audio client:load id="jyu5" />

### 4.2 母音 **eo** と **oe**

<Alert color="success">
  **oe** は **e** の円唇バージョンで、IPAは[œ]です。多くの話者は英語の bird の ir や fur の ur のように聞こえると言います。
</Alert>

**eo** は **oe** の短いバージョンで、口の開きが **oe** より小さく、IPAは[ɵ]です。以下の例を聞いてください。

- 率 seot1 <Audio client:load id="seot1" />
- 潤 jeon6 <Audio client:load id="jeon6" />
- 鋸 goe3 <Audio client:load id="goe3" />
- 削 soek3 <Audio client:load id="soek3" />
- 讓 joeng6 <Audio client:load id="joeng6" />

**eo** と **oe** の違いが分からなくても大丈夫です。多くのネイティブスピーカーでさえ区別できません。これら2つの音を混同しても完全に理解されます。

### 4.3 子音 **z** と **c**

<Alert color="success">
  **z** は日本語の「ざ」行の子音に近いですが、IPAは[t͡s]（および異音[t͡ʃ]）です。
</Alert>

**c** は **z** の有気バージョンです。IPAは[t͡sʰ]（および異音[t͡ʃʰ]）です。

| **z**                                   | **c**                                   |
| --------------------------------------- | --------------------------------------- |
| 知 zi1 <Audio client:load id="zi1" />   | 詞 ci4 <Audio client:load id="ci4" />   |
| 借 ze3 <Audio client:load id="ze3" />   | 車 ce1 <Audio client:load id="ce1" />   |
| 左 zo2 <Audio client:load id="zo2" />   | 坐 co5 <Audio client:load id="co5" />   |
| 炸 zaa3 <Audio client:load id="zaa3" /> | 查 caa4 <Audio client:load id="caa4" /> |

### 4.4 子音 **ng**

<Alert color="success">
  日本語では、ng の音（IPA [ŋ]）は「ん」の一部として現れますが、広東語では音節の最初と最後の両方に現れることができます。以下の例を聞いてください。**ng-** が音節の最初の子音として機能しています。
</Alert>

- ngaa4 牙 <Audio client:load id="ngaa4" />
- ngaang6 硬 <Audio client:load id="ngaang6" />
- ngo5 我 <Audio client:load id="ngo5" />
- ngoi6 外 <Audio client:load id="ngoi6" />

**ng-** の発音が難しいかもしれませんが、心配しないでください。多くのネイティブスピーカーも発音できませんが、完全に理解されます。

## 5 声調

これは日本語話者にとってまったく新しい領域になります。広東語は声調言語なので、単語の音の高さが重要です。間違った音の高さを使うと、単語の意味が変わってしまうことがあります。

すべてを区別できなくても心配しないでください。実生活では十分に理解されます。練習するには、[声調練習](/jp/blog/tones)のページをチェックしてください。すべての広東語声調の包括的な練習ができます。

広東語の声調を練習するもう一つの楽しく効果的な方法は、広東語の歌を通じてです。詳細は[広東語の歌で広東語の声調を練習する](/jp/blog/song/)をご覧ください。
