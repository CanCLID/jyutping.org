粤拼嘅设计大部分同普通话拼音（即《汉语拼音》，下称「普拼」）相同，只需读完两者主要区别，就可以掌握粤拼同粤拼打字。

<Alert title="写法区别 & 音系区别">

粤拼同普拼嘅区别分两类：写法区别同音系区别。

- **写法区别** 即呢个发音喺普通话同粤语入边都有，只不过喺两种拼音入边用唔同嘅字母来代表，呢类区别一睇就识。
- **音系区别** 即普通话冇呢个音，得粤语独有，呢类往往都系普通话母语者学粤语嘅重点难点。

</Alert>

> 本文讲解发音个阵会用到国际音标（IPA），如果读者唔熟国际音标都冇所谓，直接揿个 掣听例字嘅录音就可以感受到个发音嘞。
>
> 本文所有粤拼都用粗体显示，每只例字拼音后面嘅数字 1 - 6 系声调，读者可以暂时先忽略，或者直接跳到 [2.8 聲調](#28-声调)一节了解完声调再开始读本文。

## 1. 写法区别

### 1.1 普拼 y → 粤拼 j

<Alert color="success">

普拼嘅 y 喺粤拼入边写成 **j**。之所以写成 **j** 系出于两个考虑：一系呢个音国际音标入边就系写成 [j]（普拼嘅 y 系英文嘅写法）；二系如果写成 y，会同韵母 **yu** 捞乱（详细解释可见本站嘅[关于 j 同 y](/blog/scheme/#关于-j-同-y)）。

</Alert>

| 例字                               | ✔️ 正确写法 | ❌ 错误写法 |
| ---------------------------------- | ----------- | ----------- |
| 一 <Audio client:load id="jat1" /> | jat1        | yat1        |
| 以 <Audio client:load id="ji5" />  | ji5         | yi5         |
| 夜 <Audio client:load id="je6" />  | je6         | ye6         |

### 1.2 普拼 ü → 粤拼 yu

<Alert color="success">

呢个音嘅国际音标系 [y]。呢个韵母虽然系单元音，但喺粤拼入边用咗 **yu** 两个字母来表示，噉系为咗避免好似普通话嘅 ü 噉要用到附加符号，另一方面都系前文提到嘅，防止同 **j** 捞乱。

</Alert>

| 例字                                | ✔️ 正确写法 | ❌ 错误写法 |
| ----------------------------------- | ----------- | ----------- |
| 猪 <Audio client:load id="zyu1" />  | zyu1        | zü1         |
| 村 <Audio client:load id="cyun1" /> | cyun1       | cün1        |
| 雨<Audio client:load id="jyu5" />   | jyu5        | jü5 / yu5   |

### 1.3 普拼 gu-、ku- → 粤拼 gw-、kw-

<Alert color="success">

普拼嘅 gu-、ku-，喺粤拼入边用 **gw-**、**kw-** 来表示。不过要注意，gu、gun 仲系 gu、gun。

</Alert>

| 例字                                 | ✔️ 正确写法 | ❌ 错误写法 |
| ------------------------------------ | ----------- | ----------- |
| 广 <Audio client:load id="gwong2" /> | gwong2      | guong2      |
| 狂 <Audio client:load id="kwong4" /> | kwong4      | kuong4      |
| 昆 <Audio client:load id="kwan1" />  | kwan1       | kuan1       |
| 古 <Audio client:load id="gu2" />    | gu2         | gwu2 / gw2  |

### 1.4 普拼 ong → 粤拼 ung

<Alert color="success">

普拼中嘅 ong 同粤拼嘅 **ung** 发音系一样嘅，国际音标都系 [ʊŋ]，而粤拼中嘅 **ong** 系普通话冇嘅发音，国际音标系 [ɔŋ]。

</Alert>

| ung                                       | ong                                       |
| ----------------------------------------- | ----------------------------------------- |
| 钟 zung1 <Audio client:load id="zung1" /> | 庄 zong1 <Audio client:load id="zong1" /> |
| 公 gung1 <Audio client:load id="gung1" /> | 江 gong1 <Audio client:load id="gong1" /> |

### 1.5 普拼 ao → 粤拼 aau

<Alert color="success">

粤拼以 u 而非 o 结尾，呢个都系遵循国际音标嘅写法。

</Alert>

| 例字                                | ✔️ 正确写法 | ❌ 错误写法  |
| ----------------------------------- | ----------- | ------------ |
| 考 <Audio client:load id="haau2" /> | haau2       | hao2 / haao2 |
| 包 <Audio client:load id="baau1" /> | baau1       | bao1 / baao1 |

> 关于粤拼中 a 同 aa 嘅区别下面会讲解。

## 2. 音系区别

### 2.1 韵腹 aa 同 a

<Alert color="success">

普拼嘅 a 等同于粤拼嘅 **aa**，国际音标系 [a]。而粤拼中嘅 **a** 音喺普通话入边冇。佢嘅国际音标系 [ɐ]。**aa** 同 **a** 系两个不同嘅韵腹，唔好捞乱。读者既然识粤语，噉睇下下面几个例字就知道个区别嘞：

</Alert>

| aa                                        | a                                       |
| ----------------------------------------- | --------------------------------------- |
| 三 saam1 <Audio client:load id="saam1" /> | 心 sam1 <Audio client:load id="sam1" /> |
| 山 saan1 <Audio client:load id="saan1" /> | 身 san1 <Audio client:load id="san1" /> |
| 稍 saau2 <Audio client:load id="saau2" /> | 手 sau2 <Audio client:load id="sau2" /> |

### 2.2 韵腹 eo、oe

<Alert color="success">

呢两个韵母嘅国际音标分别系 [ɵ]、[œ]，普通话入边都冇呢两个音。

</Alert>

| eo                                        | oe                                          |
| ----------------------------------------- | ------------------------------------------- |
| 率 seot1 <Audio client:load id="seot1" /> | 削 soek3 <Audio client:load id="soek3" />   |
| 润 jeon6 <Audio client:load id="jeon6" /> | 让 joeng6 <Audio client:load id="joeng6" /> |

> 大家可能会觉得，点解要将 **eo** 同 **oe** 分开？呢两个唔系同一个音来嘅咩？其实唔系。**eot6** 系「术」字嘅韵母，但系 **oet6** <Audio client:load id="oet6" /> 字就系另一个音嘞。「食完饭 **oet6** <Audio client:load id="oet6" /> 一啖气出来」，就系呢个韵母 oet6。
>
> 再例如形容赶快嘅词「**coet4** **coet2** 声」，好明显唔系「出出声」，所以呢两个系唔同嘅音来㗎。
>
> 详细嘅解释可以参考[粤拼答问](https://lshk.org/jyutping-faq)第 F 问。

### 2.3 韵母 e

<Alert color="success">

普拼嘅 e 国际音标系 [ɤ]，即系普通话入边「饿」字嘅韵母，如果佢前面接 y 或者 i 则表达另一个音 [e]，例如普通话里「夜」字的韵母。而粤拼入边 **e** 永远净代表一个发音 [ɛ]，呢个音类似英语单词 「yes」 嘅元音。

</Alert>

- 写 se2 <Audio client:load id="se2" />
- 借 ze3 <Audio client:load id="ze3" />
- 郑 zeng6 <Audio client:load id="zeng6" />
- 石 sek6 <Audio client:load id="sek6" />

### 2.4 韵母 i

<Alert color="success">

普拼入边 i 会根据拼写音节嘅唔同会代表两个唔同嘅音。例如普通话入边「资」嘅拼音系 zi，呢度 i 代表嘅音系 [z̩]，而喺「机」嘅拼音 ji 度，i 代表嘅音系 [i]。换句话讲，普拼中 zi ci si 入边嘅 i，同 ji qi xi 入边嘅 i 系两个唔同嘅 i。

粤拼嘅 **i** 情况类似， 喺大多数时候都代表发音 [i]，而拼写成 **ik** 或 **ing** 个阵发音就变成 [ɪ]。

</Alert>

| [i]                                     | [ɪ]                                       |
| --------------------------------------- | ----------------------------------------- |
| 诗 si1 <Audio client:load id="si1" />   | 识 sik1 <Audio client:load id="sik1" />   |
| 仙 sin1 <Audio client:load id="sin1" /> | 星 sing1 <Audio client:load id="sing1" /> |
| 知 zi1 <Audio client:load id="zi1" />   | 即 zik1 <Audio client:load id="zik1" />   |

### 2.5 声母 ng-

<Alert color="success">

呢个音嘅国际音标系 [ŋ]。普通话冇呢个声母，而粤语入边声母 **ng-** 同零声母 Ø- 系区分嘅。而家好多广州人都分唔出嘞，例如将「我」字发成 **o5**，但系其实个发音系 **ngo5**。（好彩输入法可以开模糊音，所以如果实在分唔到就开模糊音啦，冚唪唥打晒 **ng-** 开头就得）。

</Alert>

| ng-                                         | Ø-                                      |
| ------------------------------------------- | --------------------------------------- |
| 岸 ngon6 <Audio client:load id="ngon6" />   | 按 on3 <Audio client:load id="on3" />   |
| 雁 ngaan6 <Audio client:load id="ngaan6" /> | 晏 aan3 <Audio client:load id="aan3" /> |

### 2.6 韵尾 -m

<Alert color="success">

普通话入边鼻音韵尾净得 -n 同 -ng（前鼻音同后鼻音），冇 -m。而粤语保留咗 **-m**、**-n**、**-ng** 三种鼻音韵尾。

</Alert>

| -m                                        | -n                                        | -ng                                         |
| ----------------------------------------- | ----------------------------------------- | ------------------------------------------- |
| 三 saam1 <Audio client:load id="saam1" /> | 山 saan1 <Audio client:load id="saan1" /> | 生 saang1 <Audio client:load id="saang1" /> |
| 含 ham4 <Audio client:load id="ham4" />   | 痕 han4 <Audio client:load id="han4" />   | 行 hang4<Audio client:load id="hang4" />    |
| 尖 zim1 <Audio client:load id="zim1" />   | 煎 zin1 <Audio client:load id="zin1" />   | 精 zing1 <Audio client:load id="zing1" />   |

### 2.7 韵尾 -p、-t、-k

<Alert color="success">

呢三个韵尾就系传统意义上嘅「入声」。普通话冇入声，但系粤语就保留嗮，所以既然读者识粤语，下面几只字嘅发音应该都好容易区分得出来。

</Alert>

| -p                                        | -t                                      | -k                                      |
| ----------------------------------------- | --------------------------------------- | --------------------------------------- |
| 湿 sap1 <Audio client:load id="sap1" />   | 失 sat1 <Audio client:load id="sat1" /> | 塞 sak1 <Audio client:load id="sak1" /> |
| 接 zip3 <Audio client:load id="zip3" />   | 节 zit3 <Audio client:load id="zit3" /> | 脊 zek3 <Audio client:load id="zek3" /> |
| 集 zaap6 <Audio client:load id="zaap6" /> | 活 wut6 <Audio client:load id="wut6" /> | 六 luk6 <Audio client:load id="luk6" /> |

### 2.8 声调

<Alert title="提醒" color="dark">

如果净系想学识用粤拼打字，可以**直接跳过本节**。粤拼打字唔需要打声调，不过如果打埋声调嘅话可以提高输入速度。

</Alert>

> 可能会有人问，点解我觉得广州话唔止六个声调嘅？例如「私人」同「诗人」嘅音唔一样㖞，点解都系 si1？呢个系因为粤语广州话入边嘅第一声的确已经分化咗成两个声调，有啲字例如「诗」读成 高平调 55，有啲字例如「私」读成高降调 53。但系喺粤拼入边我哋**唔做区分，全部当成第一声**。

> 详细嘅解释可以参考[粤拼答问](https://www.lshk.org/faq-on-jyutping)第 I 问。

### 2.9 普通话特有、粤语冇嘅音

<Alert color="success">

粤语唔分平翘舌音，所以粤拼入边冇 zh ch sh r 呢四个声母，另外粤语冇普拼中嘅 j q x 音（粤拼嘅 j 等于普拼嘅 y）。

</Alert>

## 3. 搞掂！

恭喜你读到呢度，粤拼同普通话拼音嘅区别就咁多，其他字母代表嘅发音，例如 b p m f、d t n l、g k h、z c s w 喺普拼同粤拼入边都系一样嘅。要查睇成个粤拼方案嘅完整声韵母表，请见本站嘅[ LSHK 粤拼方案](/jyutping)。而家先来做几个小练习，睇下你学识打粤拼未：

### 3.1 练习

写出下面几个词嘅粤语拼音，可以唔理个声调：

1. 粤语拼音
2. 广州话
3. 稍等
4. 你哋
5. 我同佢
6. 香港
7. 语言学

#### 答案（可以忽略个声调）

1. jyut6 jyu5 ping3 jam1
2. gwong2 zau1 waa2
3. saau2 dang2
4. nei5 dei6
5. ngo5 tung4 keoi5
6. hoeng1 gong2
7. jyu5 jin4 hok6
