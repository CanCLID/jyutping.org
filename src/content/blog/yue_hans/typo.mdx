粤文一般使用传统汉字做书写系统，因此冇简化字中多字合并嘅问题。关于大陆简化字嘅归并问题，请参见[简化合并字表](http://www.kaom.net/font_jianhua_hbz.php)同[简化同形字表](http://www.kaom.net/font_jianhua_txz.php)。

## 常见错别字表同推荐用字方案

<Alert title="说明">

表中部分字有唔止一个建议用字，主要考虑到粤语区唔同地域用字习惯，表示呢几种用字皆可。

</Alert>

| 粤拼                  | 常见混用写法 | 释义 & 判别标准                                                                                                                                                                                                                                                                                               |
| --------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| bei2                  | 比畀俾       | 三只字皆读 bei2，其中<ul><li>「比」义为【比较】</li><li> 「畀」义为【给予】、【被动】</li><li> 「俾」罕用字，一般用于译名例如「俾斯麦」或者成语</li></ul>                                                                                                                                                     |
| dei2/dei6             | 地哋         | 两只字都有 dei6 读音，其中：<ul><li>「哋」仅用于复数人称代词，例如「我哋」「佢哋」</li><li>「地」可用于「地面」「地点」等，另有一发音 dei2，用喺形容词度，例如「神神地」「傻傻地」</li></ul>                                                                                                                  |
| di1/dit1              | Dd 滴尐啲    | 读作 di1 或 dit1 ，意思系【少量】，一律写作「啲」。其余都系非规范写法。                                                                                                                                                                                                                                       |
| duk1                  | 笃督㞘㧻涿𡰪 | 全部都读 duk1，其中<ul><li>「㞘」指【最尾】【末端】，例如「行到㞘」。另可写作「𡰪」</li><li>「㧻」系动词，指【刺】【戳】</li><li>「涿」系量词，指「一涿屎」「一涿尿」</li><li>「督」用于「监督」、「都督」等</li><li>「笃」用于「笃信」、「笃定」等</li></ul>                                                 |
| fan3                  | 瞓训         | 读音均为 fan3，其中<ul><li>「瞓」意思系【睡眠】，例如「瞓觉」</li><li> 「训」用于「训练」「训教」等</li></ul>                                                                                                                                                                                                 |
| faan1                 | 翻返番       | 皆有 faan1 读音，其中<ul><li>「翻」义为【越过】【攀爬】</li><li> 「返」义为【返回】、【前往】【回复体标记】，例如「返工」、「整返好佢」，亦有另一读音 faan2。</li><li> 「番」意思有【次数】、【西方嘅】</li></ul>                                                                                             |
| gam2/gam3             | 噉咁         | 两字皆有【如此】嘅意思，其中<ul><li> 「噉」读 gam2，代词，例如「噉样嘅话」</li><li> 「咁」读 gam3，副词，例如「咁犀利」</li></ul>                                                                                                                                                                             |
| gau6                  | 旧嚿䊆       | 两字皆读 gau6，其中<ul><li>「嚿」系量词，意思系【块】。另有写作「䊆」。</li><li> 「旧」意思系【古老】</li></ul>                                                                                                                                                                                               |
| ge3                   | 既嘅噶       | <ul><li>「嘅」读 ge3，系属格标记，例如「我嘅书」。</li><li> 「既」读 gei3，用例「既然」</li><li> 「噶」读 got3，系拟声词</li></ul>                                                                                                                                                                            |
| go2/go3/gwo2          | 𠮶个果       | <ul><li>「𠮶」系远指代词，读 go2，例如「𠮶个人」</li><li> 「个」读 go3，用例「一个人」</li><li> 「果」读 gwo2，用例「果实」「水果」</li></ul>                                                                                                                                                                 |
| haa2/haa5             | 吓下         | <ul><li>「吓」读 haa2，仅用作语气词，例如「吓？系真嘎？」「系噉嘅咩吓？」</li><li>「下」有两个发音，读 haa6 个阵表示方向，例如「下面」「下方」；读 haa5 个阵表示次数，例如「畀我睇下」「试下先」 </li></ul>                                                                                                   |
| hai6                  | 系系喺系     | <ul><li>「系」读 hai6， 用作谓语</li><li> 「系」读 hai6，用例「系列」</li><li> 「喺」读 hai2，表示所处时间方位，意思系【位于】，用例「我喺度」</li><li> 「系」读 hai6，用例「联系」。</li></ul>                                                                                                               |
| je5                   | 野嘢         | 读音都系 je5，其中<ul><li> 「嘢」系名词，指【事物】</li><li> 「野」系形容词，用例「野外」</li></ul>                                                                                                                                                                                                           |
| lai4/lei4/loi4        | 来嚟黎       | <ul><li> 来：读 loi4 ，用例「将来」「未来」</li><li> 嚟：读 lai4 或者 lei4，例如「过嚟」</li><li>黎：读 lai4，用于姓氏或者「黎明」等</li></ul>                                                                                                                                                                |
| mou4/mou5             | 无冇         | 两者皆为否定词，其中<ul><li>「无」读 mou4</li><li>「冇」读 mou5</li></ul>                                                                                                                                                                                                                                     |
| nam2/lam3             | 谂惗冧       | <ul><li>「谂」读 nam2，意思系【思考】【想】。</li><li> 「惗」读 nam2，系「谂」嘅异体字，唔推荐使用</li><li> 「冧」读 lam3，指【倒塌】，例如「冧屋」「冧楼」</li></ul>                                                                                                                                         |
| sai2                  | 使驶洗       | 三只字都读 sai2，其中<ul><li> 「使」用于「使用」，例如「使钱」「唔使客气」；另外有 si3 嘅读音，用于「使者」「信使」「大使馆」等</li><li> 「驶」用于「驾驶」</li><li> 「洗」指【清洁】，注意「使钱」同「洗钱」同音唔同义</li></ul>                                                                             |
| saai3                 | 晒晒嗮哂     | <ul><li>「晒」读 saai3，指「晒太阳」</li><li> 「晒」读 saai3，系助词，指【全部】例如「食晒饭」</li><li> 「嗮」读 saai3，一般同「晒」意思相同，但系唔推荐使用</li><li> 「哂」读 can2，指【讥笑】，例如「哂笑」 </li></ul>                                                                                      |
| zo2                   | 左唨咗       | 读音皆为 zo2，其中<ul><li>「左」指方向，例如「左手」</li><li> 「咗」完成体标记，例如「食咗饭」</li><li> 「唨」同「咗」意思相同，但系唔推荐用呢只字 </li></ul>                                                                                                                                                 |
| zyu3/zyu6/zoek3/zoek6 | 著住着       | <ul><li> 「著」读 zyu3，用例「著作」「著名」</li><li> 「住」读 zyu6，系助动词，例如「袋住先」；亦有「居住」嘅意思</li><li> 「着」有两个发音，读 zoek3 𠮶阵系动词，例如「着衫」；读 zoek6 𠮶阵有两个意思，第一个系官话入边嘅进行体标记，例如「唱着歌」；第二个系【启动】嘅意思，例如「着火」「着色」</li></ul> |
